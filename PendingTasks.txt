Feature 4 — Day Complete Screen

1. In your Canvas hierarchy, create a new panel: Right-click Canvas → UI → Panel, name it DayCompletePanel
2. Add a CanvasGroup component to DayComplet
3. Inside it, add two TextMeshPro - Text (UI) children:
  - TitleText — large, centered (e.g., font
  - SubtitleText — smaller, centered below
4. Add a Button child → label it "Continue"
5. Add component DayCompleteUI to DayCompletePanel
6. Wire up in Inspector: drag TitleText → Tiitle Text
7. On the Button's OnClick() event: drag DayCompletePanel → select DayCompleteUI.OnDismiss
8. Save the scene

The panel will auto-hide on Awake via the Cath missions are completed optimally.
